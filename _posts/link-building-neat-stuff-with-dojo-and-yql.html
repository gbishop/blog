Interesting library for accessing data from Javascript.

Enterprise Dojo

YQL is defined by Yahoo! as: ‘an expressive SQL-like language that lets you query, filter, and join data across Web services.’ In a nutshell, YQL lets you query for interesting data that can be easily consumed by a web app.
