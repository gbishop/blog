I looked high and low and couldn't find a working example of connecting a dojox.grid.DataGrid to a dojox.data.JsonRestStore so I puzzled out parts of it myself. This code probably has errors and omissions but it seems to work for me with dojo 1.4.1.

I would appreciate any corrections or pointers to other examples from those of you who know better. I haven't included any error checking and the references to items by key break down after deletions because of the simple minded implementation of the "database". I wrote the little server using tornado but I'm not using any special features of that package.
